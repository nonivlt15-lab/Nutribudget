# Nutribudget

Track your nutrition and your budget together in one place.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- **Daily snapshot** – Track money spent and calories in one dashboard
- **Add entries** – Log meals and purchases with amount and calories
- **Progress tracking** – Visual progress bars and ring for budget and calorie goals
- **Dark theme** – Built-in dark mode with green accent colors

## Tech Stack

- Next.js 15
- React 19
- Tailwind CSS v4
- TypeScript
